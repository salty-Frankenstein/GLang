# 彁语言

## Keywords
`彁`: for `(`  
`恷`: for `)`  

### Values
`碵`: for `nil`  
`粐`: for `#t`  
`糘`: for `#f`  


### Operators
`垉`: for `+`  
`垈`: for `-`  
`墸`: for `*`  
`壥`: for `/`  
`暃`: for `==`  
`橸`: for `null?`  

### Lambda Functions
`彁汢<Name><Expr>恷`  

### Assign
`彁粭<Name><Expr>恷`  

### Block
`彁熕<Expr1><Expr2>...恷`  

### if expression
`彁挧<condition><then expr><else expr>恷`  

### putChar
`彁椦<Expr>恷`
